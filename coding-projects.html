Damian's Coding Projects

GA Project-1: The Jeopardy Game - For this first coding 'project' experience, I opted to undertake the Jeopardy game. I created a basic, single-player version of the popular TV game show. When a player comes to the site, my hope is the gamer finds a fun-filled, intuitive user experience.

GA Project-4: Django-React App - This project's objective was to develop a web application using React on the frontend with a Django REST Framework (DRF) backend connection via API calls utilizing Axios, the promise based HTTP client. The app allows users to track pantry items and other perishable foodstuffs within a 'virtual' online pantry. It also leverages a 3rd party web API so PantryApp users can obtain listings of nearby farmers markets where they might replenish their perishables.
